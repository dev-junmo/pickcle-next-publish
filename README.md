# Conpick User Web

## Commands

### 설치

```sh
  npm ci
```

### 개발모드 실행

```sh
  npm run dev
```

### 타입 체크, 문법 검사

```sh
  npm run lint
```

### 빌드

```sh
  npm run build
```

### 빌드파일 실행

```sh
  npm start
```

### 포트 변경(9999번 예시)

```sh
  npm run dev -- --port 9999
  npm start -- --port 9999
```
